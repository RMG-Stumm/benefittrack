# BenefitTrack

Client Renewal Management application built with React + Vite.

## Setup

1. Replace `src/App.jsx` with your full BenefitTrack source code
2. Run `npm install`
3. Run `npm run dev` to test locally
4. Push to GitHub and connect to Netlify

## Deployment

This project is configured for Netlify deployment.
- Build command: `npm run build`
- Publish directory: `dist`

## How to replace App.jsx

1. Open your old working artifact in Claude
2. Click the Copy button (top right of artifact)
3. Open `src/App.jsx` in a text editor
4. Select all and replace with your copied code
5. Save the file
